Phase 1 — Create br0 Bridge (Netplan) on Each Host
DELLB Netplan Bridge Config eno1 → br0
1. Backup existing netplan config
$ sudo cp /etc/netplan/*.yaml \ /etc/netplan/backup-original.yaml
2. Find your current config file
$ ls /etc/netplan/
3. Replace with bridge config
$ sudo nano /etc/netplan/01-netcfg.yaml
Content (paste exactly):
# dellb — br0 bridge over eno1 network: version: 2 renderer: networkd ethernets: eno1: dhcp4: false dhcp6: false bridges: br0: interfaces: [eno1] addresses: - 192.168.10.3/24 routes: - to: default via: 192.168.10.1 nameservers: addresses: [192.168.10.1, 8.8.8.8] parameters: stp: false forward-delay: 0 dhcp4: false dhcp6: false
4. Apply (WARNING: SSH will drop briefly)
$ sudo netplan apply
5. Verify bridge is up
$ ip -br addr show br0
✓ Should show br0 UP with 192.168.10.3/24

Phase 2 — Switch VMs from NAT to br0 (virt-manager or virsh)
GUI virt-manager method (each VM) both hosts
Steps
1. Open virt-manager
2. Right-click VM → Open
3. Click the ⓘ (Show hardware details) icon
4. Select NIC in the left panel
5. Change Network source from
   Virtual network 'default' (NAT)
   → Bridge device: br0
6. Click Apply
7. Repeat for all 5 VMs
⚠ VM must be shut off before changing NIC config.
CLI virsh method (faster, scriptable) both hosts
Edit each VM's XML directly
$ virsh edit master-01
Find this block in the XML:
<!-- BEFORE (NAT) --> <interface type='network'> <source network='default'/> </interface>
Replace with:
<!-- AFTER (bridge) --> <interface type='bridge'> <source bridge='br0'/> <model type='virtio'/> </interface>
Repeat for all VMs
$ for vm in k8client master-01 worker-01 worker-02 spare-01; do echo "=== $vm ===" virsh dumpxml $vm | grep -A3 'interface' done
Phase 3 — Assign Static IPs Inside Each VM (Rocky Linux 9)
ROCKY9 nmcli — Set Static IP (run inside each VM via console) all VMs
Example: master-01 on dellb (192.168.10.51)
# Find connection name first $ nmcli con show # Set static IP (replace IP and con-name) $ nmcli con mod "Wired connection 1" \ ipv4.addresses 192.168.10.51/24 \ ipv4.gateway 192.168.10.1 \ ipv4.dns "192.168.10.1 8.8.8.8" \ ipv4.method manual # Apply $ nmcli con up "Wired connection 1" # Verify $ ip a show eth0
ℹ Replace the IP with the correct one per VM. Same command works on all Rocky9 VMs.
